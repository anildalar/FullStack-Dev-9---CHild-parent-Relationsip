CSS Selectors

1. Element Selector

			element{
						property:value;
			}

2. ID Selector   #
		Syntax
		element#value{
					property:value;
		}
		
		Example
		#a1{
					color:red;
		}

3. Psudo Class Selector  :

		element:hover{
		}

4. Class Selector  .   (To reduce the source code)
		
		Syntax
		
		.className{	
				property:value;
		}
		
		Ex.
		div.a{
			
		}
		
5. Attribute Selector   []
			
		Syntax
		
		[attributeName="Value"]{
				property:value;
		}

		Ex.

6. Combintion Selector   ,

		selector1,selector2,selector3...{
				property:value;
		}
